# AI Resume Analyzer and Job Recommendation System

A web-based application that analyzes resumes using NLP techniques and
recommends suitable job roles based on skill matching.

## Features
- Resume text analysis
- Skill extraction
- Job-role matching
- Simple web interface

## Technology Stack
- Backend: Python, Flask
- Frontend: HTML, CSS, JavaScript
- Dataset: CSV-based job profiles

## Applications
- Recruitment automation
- Career guidance systems
